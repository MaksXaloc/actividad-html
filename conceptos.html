Etiquetas de texto

Las etiquetas de texto en HTML permiten dar formato al contenido, como resaltar palabras importantes o enfatizar partes del texto. 


<p>Este es un párrafo</p>
<strong>Texto importante</strong>
<em>Texto enfatizado</em>

Listas

Las listas permiten organizar información de forma ordenada o desordenada, facilitando la lectura del contenido. 


<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

Imágenes

Las imágenes se utilizan para mostrar contenido visual en una página web y deben incluir siempre un texto alternativo para accesibilidad. 


<img src="imagen.jpg" alt="Descripción de la imagen">

Tablas

Las tablas permiten presentar datos de forma estructurada mediante filas y columnas. 


<table>
    <thead>
        <tr><th>Nombre</th></tr>
    </thead>
</table>

Formularios

Los formularios permiten recoger información del usuario mediante diferentes campos de entrada. 


<form>
    <input type="text">
    <button>Enviar</button>
</form>
