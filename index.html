HTML es el lenguaje básico para crear páginas web. Permite estructurar el contenido mediante etiquetas, facilitando la organización de textos, imágenes y enlaces de forma clara y comprensible.

Gracias a HTML podemos definir la estructura de un sitio web, separar correctamente el contenido y mejorar tanto la accesibilidad como el posicionamiento en buscadores.
